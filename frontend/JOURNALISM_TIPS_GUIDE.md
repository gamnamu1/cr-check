# 저널리즘 Tip 추가 가이드

이 문서는 `app/page.tsx`의 `JOURNALISM_TIPS` 배열에 새로운 문구를 추가하는 방법을 설명합니다.

## 현재 상태

- **문제적 보도 관행**: 10개 (목표 30개)
- **바람직한 저널리즘 원칙**: 5개 (목표 15개)
- **총 Tip 개수**: 15개

## 추가 방법

### 1. 파일 위치
`frontend/app/page.tsx` 파일의 6-32번째 줄에 있는 `JOURNALISM_TIPS` 객체를 수정합니다.

### 2. 문제적 보도 관행 추가

```typescript
problematicPractices: [
  // 기존 10개...

  // 새로운 문구 추가
  "새로운 문제적 관행 설명...",
],
```

### 3. 저널리즘 원칙 추가

```typescript
principles: [
  // 기존 5개...

  // 새로운 문구 추가
  "새로운 저널리즘 원칙 설명...",
],
```

## 문구 작성 가이드

### 좋은 문구 예시
✅ **간결하고 명확**: "따옴표 저널리즘이란? 취재원의 말을 검증 없이 그대로 인용하여 책임을 회피하는 보도 행태를 말합니다."
✅ **구체적**: "무주체 피동형 표현('~로 알려졌다', '~로 전해졌다')은 정보 출처를 숨겨 책임을 회피하는 문제적 표현입니다."
✅ **교육적**: "균형 보도: 대립하는 관점을 공정하게 전달하여 독자가 스스로 판단할 수 있게 합니다."

### 피해야 할 문구
❌ 너무 긴 문구 (200자 이상)
❌ 전문 용어만 나열
❌ 부정적이거나 비난하는 톤
