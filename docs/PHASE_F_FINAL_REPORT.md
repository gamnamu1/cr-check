### 해석 (interpretation_note)

이 집계 결과는 파이프라인의 절대 성능 지표가 아닌, 프로덕션 동작 관찰
일지이다. Reserved Test Set의 label은 신문윤리위 결정과 Gamnamu 큐레이션
판단을 반영하며, 모델의 상이한 판단이 반드시 오류를 의미하지 않는다.
시민 사용자의 비판적 독해가 최종 판단 레이어임을 전제한다.

정량 수치의 구조적 원인:
- **Recall 0.60의 상한**: 매핑 사전 커버리지 제약. 63 relations만 등록되어
  있어 언론윤리헌장 tier 1~2 상위 원칙(Sonnet이 프롬프트 지시대로 1~2회
  확장 인용) 등이 unmapped로 빠진다. 20개 unmapped 마커 중 14개가 tier 1~2.
- **Precision 0.28의 구조적 한계**: label은 1~2개 대분류만 지정하나 Sonnet은
  3~7개 대분류에 걸친 다면 분석을 수행. 예: A-08의 `expected=[1-2]` vs
  `detected=[1-1, 1-2, 1-3, 1-4, 1-6, 1-7, 1-8]`. "오탐"이 아닌 "더 넓게 본
  결과".
- **TN 100%**: 과도 탐지 경향에도 TN 5건 모두 탐지 0건으로 정확 판정.
  시민 사용 안전성의 핵심 지표.

## 4. 정성적 관찰 (Gamnamu 직접 검토)

Sonnet 4.5 + 4.6 파이프라인이 reserved set 53건에서 보인 **정성적** 특징:

- **C-03 전남일보 (이달의 기자상 수상작)**: 정직한 침묵 — 수상 보도에 대해
  무리한 위반 탐지를 하지 않고 탐지 0건으로 내보냄.
- **E-08 미국 대선 통신사 출처 기사 (정치적 맥락)**: 모델의 자율적 절제 —
  외국 정치 이슈에 대해 한국 언론 규범의 과잉 적용을 피함.
- **A2-02 사진 모자이크 부재 이슈**: 텍스트 파이프라인의 합리적 한계 노출 —
  이미지 검증은 현 아키텍처 범위 밖이므로 텍스트만으로는 판정 불가.
- **A2-04 / A2-10 제목 선정성 / 재난 부적절 표현**: 본문은 통과하나 제목과
  소제목의 부적절 표현 미탐지. Phase G 개선 후보 (발견 5).
- **"제목 임의 수정 후 청정 판정" 다수 케이스**: 모델이 제목 조작을 탐지해
  원 기사 문제를 정확히 짚지 못함. 모델의 행동 **정직성** 지표 — 없는 문제를
  만들지 않고, 주어진 텍스트만을 근거로 판단.

→ **TN 100% + 정직한 침묵 + 과잉 적용 거절**의 조합이 시민 도구로서의 안전
   프로파일을 형성.

## 8. 결론

Phase F 공식 완료. CR-Check은 베타 공개 가능한 **안전 프로파일**을 갖추었다:

- **TN 100%** — 문제 없는 기사를 문제있다고 지목하지 않음
- **정직한 침묵** — 증거 없는 맥락(이달의 기자상, 외국 정치)에서 탐지 0건
- **과잉 적용 거절** — 한국 규범의 외국 맥락 무리한 확장 없음
- **격리 원칙 완수** — Pool 미접근, 블라인드 로딩, 실행/채점 분리

정밀도 개선의 진짜 길은 모델 튜닝이 아닌 **데이터 레이어 보강**에 있음을
Phase F가 명확히 식별했다. `pattern_ethics_relations` 매핑이 14개 source 중
주로 2개(신문윤리실천요강 + 언론윤리헌장)에 편중된 상태에서, Sonnet의 인용
행동(69.3% + 25.2%)도 자연스럽게 그 편중을 따라간다. 두 layer의 불균형을
해소하는 것이 F1을 구조적으로 끌어올리는 유일한 지렛대이며, 모델 교체나
프롬프트 튜닝으로는 도달할 수 없는 천장이다.

**Phase G의 1순위 작업**: 발견 1 + 발견 2 병행 해소
(매핑 확장 + 임베딩 재분포). 이 둘이 해결되면 발견 3은 자동 해소되며,
recall의 구조적 상한이 걷히고 precision의 맥락 가중치 조정 여지가 생긴다.

Phase G 부차 작업: 발견 4(파서 커버리지), 발견 5(제목 레이어 검사).

Phase F는 모델이 잘한 것과 데이터가 부족한 것을 분리해 보여주었다.
파이프라인은 준비되어 있다 — 필요한 것은 시간과 큐레이션이다.
