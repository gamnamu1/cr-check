# 골든 데이터셋 — 1-8 디지털 영역 보강 리서치 브리프

> **작성**: 2026-03-23
> **작성자**: Gamnamu + Claude
> **목적**: 외부 리서처(Perplexity, Manus, Gemini 등)에게 1-8 디지털 영역의 **실제 문제 기사**를 찾아달라고 요청하기 위한 브리프
> **핵심 원칙**: "디지털에 대한 비평 기사"가 아니라, **디지털 윤리를 위반한 실제 뉴스 기사 원본**을 찾는 것

---

## 0. 배경

CR-Check 골든 데이터셋 후보 풀 105건 중, 1-8 디지털 영역은 **3건**으로 유일하게 목표(4건+)에 미달합니다.
기존 3건(B2-01~03)은 "디지털 환경에 근접한 사례"이지, 순수한 디지털 윤리 위반 기사가 아닙니다.

**필요한 것**: 아래에 정의된 4가지 유형 중 **최소 2~3건**의 실제 기사 (기사 제목, 언론사, 날짜, URL 특정)

---

## 1. 찾아야 할 기사 유형 (우선순위순)

### 유형 A: AI 생성 기사의 사실 오류 (최우선)

**정의**: 생성형 AI로 작성하거나 AI 보조를 받아 작성한 기사에서 **사실관계 오류**가 발생한 사례.

**구체적 단서**:
- 2025년 4월경, 다수 국내 온라인 매체가 트럼프 현직 대통령을 "전 대통령"으로 잘못 표기 (ChatGPT 환각 현상이 원인)
- 네이버에서 "트럼프 전 대통령" 검색 시 2025년 1월 이후 작성된 오보 기사 발견 가능
- EBN뉴스 2025.04.10 보도에서 "수십 건" 확인된다고 언급

**검색 가이드**:
- 네이버 뉴스에서 `"트럼프 전 대통령"` 검색, 기간 2025.01.20~2025.06.30 필터
- 국제면·경제면·온라인 전용 매체에 집중
- AI 어시스턴트 사용 표기가 있거나, 뉴스 하단에 "AI의 도움을 받아 작성" 문구가 있는 기사 우선

**보고 형식**:
- 기사 제목, 언론사, 게시일, URL
- 오류 내용 (예: "트럼프 전 대통령이 관세를..." → 현직인데 전직으로 표기)
- AI 작성 표기 유무
- 정정 여부

**CR-Check 분류**: 1-8 디지털 (AI 생성 기사 검증 실패) + 1-1 진실성 (오보)

---

### 유형 B: 포털 어뷰징 / 기사 제목 낚시 (높음)

**정의**: 포털 사이트(네이버·다음) 상위 노출을 목적으로 기사 제목과 본문 내용이 **현저히 불일치**하는 사례. 또는 동일 사안에 대해 제목만 미세하게 변경하여 **대량 반복 전송**한 사례.

**구체적 유형** (하나만 해당해도 됨):
1. **제목-본문 불일치**: 제목은 자극적이나 본문에는 해당 내용이 없거나 미미한 경우
2. **낚시성 썸네일**: 기사 내용과 무관한 자극적 이미지를 대표 이미지로 사용
3. **기사 대량 재활용**: "~재조명", "~화제" 등으로 기존 기사를 반복 전송 (포털 트래픽 유도)
4. **실시간 검색어 어뷰징**: 인기 키워드를 억지로 끼워넣은 기사

**검색 가이드**:
- `신문윤리위원회 제목 원칙 위반` 또는 `인터넷신문윤리위원회 낚시성 기사 제재`
- `네이버 뉴스 낚시 기사 제재 사례`
- inec.or.kr (인터넷신문윤리위원회) 심의 결과에서 '제목의 원칙' 위반 건 검색
- 시기: 2023~2025년 사이

**보고 형식**:
- 기사 제목, 언론사, 게시일, URL (가능하면 네이버 뉴스 URL)
- 제목과 본문의 불일치 내용 설명
- 심의기관 제재 여부 (있으면)

**CR-Check 분류**: 1-8 디지털 (포털 어뷰징) + 1-7 언어 (제목 원칙 위반)

---

### 유형 C: 클릭베이트 / 자극적 기사 제목 (중간)

**정의**: 온라인 트래픽 유도를 위해 **과도하게 자극적·선정적인 제목**을 사용한 사례. 포털 어뷰징(유형 B)과 겹칠 수 있으나, 여기서는 특히 **"~충격", "~경악", "~발칵", "~헉"** 등 감탄사형 제목이나, **"A인 줄 알았더니 B였다"** 식의 미스디렉션 제목에 집중.

**구체적 단서**:
- 인터넷신문윤리위원회(inec.or.kr)가 매년 '낚시성 기사' 관련 심의 결과를 발표함
- 한국언론진흥재단 "뉴스 신뢰도 조사"에서 낚시성 기사가 매년 1위 문제점으로 꼽힘

**검색 가이드**:
- `inec.or.kr 낚시성 기사 심의` 또는 `인터넷신문윤리위원회 2024 2025 심의 결과`
- `신문윤리위원회 제목의 원칙 경고` (ikpec.or.kr)

**보고 형식**: 유형 B와 동일

**CR-Check 분류**: 1-8 디지털 (클릭베이트) + 1-7 언어 (제목 원칙)

---

### 유형 D: AI 생성 콘텐츠 투명성 미고지 (보통)

**정의**: AI로 생성한 텍스트·이미지를 **기사에 사용하면서 AI 사용 사실을 독자에게 알리지 않은** 사례.

**구체적 단서**:
- 조선일보는 2024.01부터 AI 어시스턴트로 기사를 작성하며 하단에 고지 문구를 넣었으나, 초기(2023.12~2024.01) 일부 기사에서 누락 가능성
- 영남일보는 AI 이미지 생성 솔루션을 사용하면서 이미지에 AI 생성 표기를 넣지 않은 사례 보고됨
- 네이버는 2024년부터 AI 자동생성 기사에 안내 문구를 노출하기 시작 — 그 이전의 미고지 사례가 있을 수 있음

**검색 가이드**:
- `AI 기사 고지 누락` 또는 `AI 이미지 미표기 언론`
- 미디어오늘, 기자협회보의 AI 관련 보도에서 구체 사례 추적

**보고 형식**:
- 기사 제목, 언론사, 게시일, URL
- AI 사용 유형 (텍스트/이미지/영상)
- 고지 표기 여부

**CR-Check 분류**: 1-8 디지털 (AI 콘텐츠 투명성) + 1-2 투명성

## 3. 산출물 형식

각 기사를 찾으면 다음 형식으로 정리해주세요:

```
기사 제목: [제목]
언론사: [언론사명]
게시일: YYYY-MM-DD
URL: [가능하면 원문 URL]
유형: A / B / C / D (위 분류 참조)
문제 요약: [2~3문장으로 어떤 디지털 윤리 위반인지 설명]
심의 제재 여부: 있음(기관명, 결정번호) / 없음 / 미확인
AI 사용 표기: 있음 / 없음 / 해당없음
```

## 4. 우선순위 및 기대 수량

| 우선순위 | 유형 | 기대 수량 | 난이도 |
|---------|------|----------|--------|
| 1 | A (AI 오보) | 1~2건 | 중 (네이버 검색 필요) |
| 2 | B (포털 어뷰징) | 1~2건 | 상 (inec.or.kr 검색) |
| 3 | C (클릭베이트) | 1건 | 중 (심의 결과 추적) |
| 4 | D (AI 미고지) | 1건 | 상 (과거 기사 추적) |

**최소 목표**: 유형 A에서 1건 + 유형 B 또는 C에서 1건 = **2건 이상**
**이상적 목표**: 4가지 유형에서 각 1건씩 = **4건**

## 5. 중요 주의사항

1. **비평/해설 기사가 아니라 실제 문제 기사를 찾아야 합니다.** "낚시 기사가 문제다"라는 미디어 비평이 아니라, 낚시 기사 **그 자체**를 찾아주세요.
2. **방송/영상 전용 콘텐츠는 제외합니다.** CR-Check은 텍스트 기사만 분석하므로, 방송 리포트만 존재하는 사례는 불가합니다. (텍스트 기사 버전이 있으면 OK)
3. **2023~2025년 사이 기사를 우선합니다.** 너무 오래된 기사는 URL 접근이 어려울 수 있습니다.
4. **신문윤리위(ikpec.or.kr) 또는 인터넷신문윤리위(inec.or.kr) 심의 이력이 있는 기사가 이상적**이지만, 심의 이력이 없어도 됩니다. 문제 유형이 명확하면 충분합니다.
