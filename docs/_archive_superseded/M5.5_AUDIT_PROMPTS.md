■ 프로젝트 배경 (간략)

CR-Check는 AI 기반 한국 뉴스 기사 품질 분석 도구입니다.
이틀간 M1(DB 스키마) → M2(시드 데이터) → M3(임베딩+벤치마크)
→ M4(RAG 파이프라인) → M5(프롬프트 고도화+결정론적 인용)를 진행했습니다.

  기사 → 청킹(chunker) → 벡터검색(pattern_matcher)

  → 규범 조회(report_generator, DB RPC)
  → Sonnet(리포트, cite 태그 출력)
  → CitationResolver(cite→원문 치환, in-memory)
  → 최종 리포트
