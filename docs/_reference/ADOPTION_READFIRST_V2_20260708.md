② 정확한 윤리규범 인용(코드=인용 경로) ③ 날카로운 비평 리포트 재료.

비판적 재현 구분). severity 의미론: low = 독자에게 건네는 검토 질문(유보형).

정상 신호 / high 단정=경고 / 주제만으로=실패 / 유보형 제기=바람직.
v2의 medium+"읽힐 수 있다"는 정상 반응. 라벨 최종 재판정은 STEP 7-A로 이관
(2018 기사 ↔ 2026 규범의 시간 정합성 안건 포함).

## 반영 내역 (이 커밋)
- A `pattern_matcher.py`: _SONNET_SOLO_PROMPT HEAD → v2, match_patterns_solo
  에 `title: Optional[str] = None` + user_message '## 기사 제목' 블록
- B `pipeline.py`: analyze_article(..., title=None) 전달 경로
- C `main.py`: run_pipeline(article_text, title=…) — 제목 입력 공백 봉합
- D `report_generator.py`: '### 1.5 심각도와 서술 어조' 가드레일
- G `supabase/migrations/20260708000000_…`: 6-2-a → PCP-10-1(violates/strong),
  JEC-1(related_to/moderate). 적용은 apply_migration으로 별도 실행 필요.
- T3 마이그레이션 주석 정정: PCP = 신문윤리실천요강 (구 '평화통일 준칙' 오기)
