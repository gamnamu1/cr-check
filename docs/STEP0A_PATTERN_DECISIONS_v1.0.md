## 3. D 카테고리 재정의

6개 감리자 전원 일치: 인과 오류와 심층성 부족은 분리 필수.
- 인과 오류 = 잘못된 것이 있는 문제 (Commission)
- 심층성 부족 = 없어야 할 것이 없는 문제 (Omission)

심층성 항목별 처리:

| 원 항목 | 처리 |
|---|---|
| 5-1 현상 중심 파편화 (원인 단락 부재) | E로 이동 [S] |
| 5-1 후속 추적 부재 | I-트랙 |
| 5-1 대안·전망 부재 | E로 이동 [S] |
| 5-1 맥락 없는 사실 나열 | C로 이동 |
| 5-1 인과관계 단순화 | D-1로 흡수 (3-3과 동일) |

D 최종 구성: 인과 오류 + 통계 오용
